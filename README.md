# A small text based dungeon crawler TUI

## Installation

Clone the repository
```bash
git clone https://github.com/makkinga/terminal-quest.git
```

## Start the game

Run the executable to start the game
```bash
cd terminal-quest
./dungeon
```

## Goal

The goal of the game is to navigate the player `b` trough the dungeon rooms using the `hjkl` keys and collect as many coins as possible without dying.

## Obstacles

There are 3 types of obstacles in the game: Lava `~`, Spiky bushes `*`, and Spike walls `‡`. Each one will do you one heart damage.
